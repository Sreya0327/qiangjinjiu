## 游戏简介

《将进酒·暗阁》以「暗阁枢机处的秘卷」为叙事框架，带你以查案者的身份，逐卷重走沈泽川与萧驰野的命运——从茶石关的血雪、昭罪寺的寒夜，到都察院的博弈、九年旧案的翻案，直至醉倒狂歌的终局。

全书 280+ 章被拆为**十卷**，覆盖原著三大卷（上卷·我本放逐臣／下卷·又为乱世雄／终卷·醉倒狂歌中）的完整主线。每一卷都可**读案情、搜线索、做抉择、聊人物、练修为**，多种互动交织，不是单一的选项点击。

## 玩法系统

- **📜 案牍**：逐卷阅读剧情，做**办案抉择**推进故事；抉择会按当前视角积累属性、影响两位主角的好感。
- **🔍 现场勘查**：案牍中的线索不再自动到手——点击「❓待勘查」逐一**勘验**，解锁线索详情。
- **🔍 查案**：仿探案游戏的**搜证系统**——输入关键人物/物证（如「沈卫」「布防」「昭雪」）搜查解锁线索，归入**证据板**（人证／物证／动机／线索），附**线索收集进度**、**剧情推进提示**、**办案日志**；集齐一卷线索可得**关键道具**。
- **📿 修为（属性点）**：沈泽川（权谋／隐忍／武艺／人心）与萧驰野（武力／决断／谋略／义气）各一套属性；办案、推敲、对话、日常皆可积累，并可自由分配**属性点**，走自己的养成路线。
- **📅 日常**：花体力做修行活动练属性，属性突破额外得属性点；「推进一日」休整回血。
- **📇 通讯录**：右上角滑出，与角色**像发消息一样聊天**；话题**随剧情逐步解锁**，可切换沈泽川／萧驰野双视角，各有可聊之人。
- **❤ 生命值 / 🎒 道具**：顶部显示；查案集齐线索得道具，休整回血——仿生存玩法。
- **🔍 推敲**：将两条线索关联，得出推理结论。
- **🗺️ 舆图**：内嵌国风大地图（六大区域热区），**点区域直接跳转对应案卷**。
- **📖 目录 / 👥 人物 / ⏳ 时间线 / 🔗 关系**：右上角滑出面板，按卷归类、随剧情解锁。
- **👁 双视角**：全程可在沈泽川／萧驰野之间切换，界面配色、可用内容随之变化。

## 操作指南

1. 打开 `index.html`，点「**开卷**」开始（有存档则可「继续游戏」）。
2. 在「**📜 案牍**」读剧情、点「🔍现场勘查」勘线索、做「办案抉择」推进；一卷完结会解锁下一卷。
3. 去「**🔍 查案**」输入关键词搜证，集齐线索得道具。
4. 在「**📿 修为**」分配属性点，「**📅 日常**」练属性、回血。
5. 右上角「**📇 通讯录**」和角色聊天（记得切视角解锁不同的人）。
6. 「**🗺️ 舆图**」点区域可跳到对应案卷。

## 本地运行 / 部署

**本地**：直接双击 `index.html` 用浏览器打开即可（纯前端，无需服务器）。

**GitHub Pages**：将 `index.html`、`地图.html` 放入仓库根目录，在仓库 Settings → Pages 选择分支即可发布。入口为 `index.html`。

## 存档说明

游戏使用浏览器 `localStorage` 自动存档（键名 `jjj_v6`），记录当前卷、场景、属性、好感、道具、生命值、对话进度等。旧版存档会自动兼容迁移。点封面「**重新开始**」可清档重来。
