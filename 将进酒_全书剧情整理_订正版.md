# 《将进酒》· 全书剧情梳理（完整订正版）

> 原著：唐酒卿《将进酒》正文 282 章 + 番外。逐章通读后重新梳理。
> 主角：**沈泽川**（字兰舟）× **萧驰野**（二郎）。宫廷权谋 · 强强 · HE。

> ⚠️ **订正说明**：此前根据章节标题推的分卷有多处顺序错误。经逐章通读，实际脉络订正如下——「咬痕」在开篇第 4 章就已埋下；沈泽川入的是**锦衣卫**（非都察院）；秋猎是重大政变、咸德帝随后驾崩、**李建恒（天琛帝）登基**在第 28 章；中博兵败真正的黑手是**户部尚书魏怀古的军粮案**，更深处牵涉**沈卫与白茶、格达勒、光诚帝朝的旧案**；终局是——**女帝储君李剑霆薨逝**，**沈泽川登基称帝、帝号淳圣帝、改国号「大靖」**，萧驰野为离北「狼王」执掌北疆军权，皇帝大婚办在离北草野，HE。本文按原著三大卷的真实顺序梳理。

## 一、世界观 · 时间线 · 势力

**朝代年号（更迭）**：光诚帝（旧朝，沈卫旧案之源）→ **咸德帝**（在位，病弱，为太后所制）→ 咸德帝驾崩 → **李建恒（天琛帝）** 登基 → 天琛帝崩 → 储君 **李剑霆**（女帝，实为女子）中毒薨逝 → 乱局落定，**沈泽川登基称帝，帝号「淳圣帝」，改国号「大靖」**。

**权力格局**：皇权旁落，**太后**与**世家/花党**（花思谦、韩丞、孔氏、薛氏、潘氏、费氏等）把持朝政；**司礼监掌印潘如贵**为权阉。

**地理**：
- **阒都**——大周都城（明理堂、昭罪寺、南林猎场、枫山、诏狱）。
- **中博六州**——茨、茶、樊、敦、端、灯，建兴王**沈卫**旧辖；**茶石关 / 茶石天坑**为东南天险（沈卫兵败、三万军士活埋之地）。
- **离北**——北疆铁骑，萧氏世镇（鸿雁山脉、天妃阙、柳阳三大营、沙一/二/三营、边博营、常驻营）。
- **启东**——戚氏世守（戚时雨、戚竹音）。
- **厥西 / 虚海**——布政司杨诚等。
- **格达勒 / 边沙十二部**——大漠强敌（长鹫部、有熊部、青鼠部、蒙骆部等；漠三川、茶石河；首领哈森等）。

**核心悬案**：**中博兵败案**——茶石关一夜失守、三万军士活埋茶石天坑、沈卫背「通敌」自焚。真相并非通敌，而是**户部尚书魏怀古自咸德四年起，勾结厥西布政司杨诚，倒卖军粮给奚鸿轩牟利，断了沈卫的军粮**；更深一层，牵涉**沈卫与边沙女子白茶的旧姻、格达勒、光诚帝朝的陈年旧案**。为父翻案、为自己而战，是沈泽川一生的主线。

## 二、上卷 · 我本放逐臣（第 1–96 章）

### 弧一 · 落魄入京（1–8）
建兴王沈卫兵败茶石河，敦州沦陷，三万军士活埋**茶石天坑**；沈卫自焚，通敌之罪无人可问，只余唯一活口——其第八子**沈泽川**（字兰舟）。沈泽川被押入诏狱，受**杖毙**之刑（土袋压身）几死；掌印**潘如贵**奉咸德帝口谕行刑。雪地里，一名离北军官用靴尖拨正他的脸——「沈卫是你老子？」——一脚踹翻，沈泽川回头**一口咬下**：这便是**萧驰野**，也是全书的第一道「咬痕」。沈泽川大病垂死，被送入**昭罪寺**幽禁，遇被囚二十年、疯癫怨世的前太子太傅**齐惠连（齐太傅）**。

### 弧二 · 昭罪寺蛰伏（9–20）
昭罪寺看守**纪纲**（沈卫旧部、身负端州边沙烧痕、亡子纪暮为救沈泽川死于茶石）暗中教沈泽川武功；齐太傅授其帝王之术。沈泽川镣铐得解，渐得喘息，并**领了锦衣卫的差职**，就此踏入阒都棋局。他结识**奚鸿轩**（宴主、后入吏部）、**姚温玉**（字元琢，冷面谋士）、**高仲雄**（国子监学生领袖）等。小福子溺死案、国子监学潮、潘如贵收「冰敬」等一桩桩风波皆与他若隐若现地相关；他与萧驰野见面必撕咬，又在试探里生出危险的默契（「同门师兄弟」）。

### 弧三 · 秋猎政变（21–28）
十月，咸德帝抱病设**南林猎场秋猎**。猎场雨夜骤变——行刺、政变并起，海良宜怒撞潘如贵（「阉贼！勿伤吾主！」），萧驰野中箭，乔天涯、费盛等卷入。政变落定，**咸德帝随后驾崩，李建恒登基（天琛帝）**；**花思谦、潘如贵、纪雷、乔天涯等下狱，交三法司会审**。朝局第一次大洗牌。

### 弧四 · 情动与结盟（29–52）
新朝伊始，萧沈关系在校场比武、山中同宿、雪中「味道」「撕咬」「兰舟」间急速升温；萧驰野给禁军新添**火铳**，李建恒（天琛帝）请**海良宜**讲学、勤政。纪纲身世（端州边沙烧痕、与左千秋旧识）浮出。一场针对新帝的**行刺案**中，沈泽川救驾，受封**五品锦衣卫镇抚**（獬豸铜牌）。萧沈「同舟」促膝，结成一朝一野的同盟——「没有共同的敌人时，他们就是自己的敌人。」

### 弧五 · 查案与疫病（53–70）
沈泽川借**查院**（御史余小再随行）与禁军之力，彻查户部账簿、行刺案，逐一结案。萧驰野一度病倒，李建恒冒雪探望、君臣言和；奚鸿轩升吏部考功司。随后阒都**疫病**大作，昭罪寺一带煎药遮口。生死之间，萧沈**定情（共枕、云雨）**。疫后论功，圣旨下——**萧驰野任总督，沈泽川得以走出昭罪寺**。

### 弧六 · 封赏与倒魏（71–96）
天琛帝大行封赏：**萧驰野进定都侯，沈泽川越级提拔**（岑愈力荐、海良宜转向）。沈泽川借势清查旧案：奚鸿轩下狱、抄薛府（薛修卓、薛修易），并顺藤揪出**户部尚书魏怀古的军粮案**——自咸德四年起，魏怀古勾结厥西布政司杨诚倒卖军粮、经奚鸿轩发往中博六州牟利，**正是断沈卫粮草、致茶石关兵败的黑手**。花香漪与戚时雨的婚事、皇嗣旧案（秦王旧居）等线并起。上卷终于**明理堂的一场倾塌**——山雨欲来，天琛朝的裂缝已不可弥合。

## 三、下卷 · 又为乱世雄（第 97–191 章）

### 弧七 · 决裂出走（97–112）
禁军贸然进军、锦衣卫带回一麻袋人头，昭示**沈泽川、萧驰野与阒都彻底决裂**。二人叛出阒都，南下**中博**，以茨州为根基，开始「又为乱世雄」的经营。

### 弧八 · 经营中博（113–148）
萧沈软硬兼施收服茨州守将**周桂**、招抚**雷常鸣/雷惊蛰/历熊**等土匪出身的悍将，组建**茨州守备军**；沈泽川**城郊施粥**收拢民心（「白衣公子」之名传遍中博）。**姚温玉在潘府中毒**（照月郡主、潘远之手）、**九年旧案**线头浮现（第 133 章「九年」）。北面**边沙十二部**（长鹫部胡和鲁等）来袭，离北（萧既明、戚竹音、陆广白）与之周旋。朝中「**皇女**」之议初起——朝堂上第一次出现女人的位置。

### 弧九 · 敦州与旧案真相（149–191）
天琛帝新丧，太后与内阁仍推**花戚大婚**。萧驰野在离北一度**吃败仗、被降职**又重起（陆亦栀、萧既明）。中博战线推进到**敦州**（守将澹台虎、叛将颜何如），「**格达勒**」之名反复出现。第 180–181 章掀开**沈卫旧案的底**——追溯**光诚帝时期**，沈卫曾与边沙女子**白茶**做过数年「同床异梦」的夫妻，旧案深处连着格达勒与边沙。鸿雁山脉暴雪血战（哈森、戚竹音的启东援兵），下卷在边关的风雪与阒都的暗涌里逼向终盘。

## 四、终卷 · 醉倒狂歌中（第 192–282 章）

### 弧十 · 端州决战（192–252）
中博暴雪之后，战事转入**端州—洛山—茶石天坑**一线，与**边沙十二部**（青鼠部、有熊部、蒙骆部、哈森）鏖战经年。萧驰野出兵端州、离北邬子余南下、敦州澹台虎策应；尹昌的「尖刀」阵、火铳、离北群鹰、浪淘雪襟……一场场硬仗收东南、定北疆。阒都同时风云突变：**储君李剑霆**（实为女子）**中毒转危为安**（风泉尝药），**丹城案**中**潘蔺之死**牵动朝局，理财能臣**梁漼山（「崇深大人」）**登场。

### 弧十一 · 女帝身世风波（253–276）
李剑霆病中理政，**女帝身世的流言**甚嚣尘上，城门封锁也压不住。**薛修卓（字延清）**迎储君回宫、向内阁出示**先帝朱批与秦王私章**，世家阵营裂解、反转（薛修卓摘韩丞腰牌）。**戚竹音**被指「叛」，启东百年威望悬于一线；**风泉**成为李剑霆身边的关键影子。庸城旱灾、筹粮、赫连侯……内政外战同时逼向决战。

### 弧十二 · 狼鹰高殿 · 终局（277–282 + 番外）
阒都决战爆发，王宫焚毁大半；边沙覆灭于**漠三川 / 格达勒**（虹鹰旗焚），「狼鹰」折翼。朝局终盘：储君**李剑霆（女帝）薨逝**；众望所归，**沈泽川登基称帝、帝号淳圣帝、改国号「大靖」**——「此身虽是大靖身，可此心仍是大周心」。太后与花党退场、世家或倾覆或收服；大靖三位异姓王，戚竹音虎踞启东、萧既明镇守离北，而**萧驰野为离北「狼王」、执掌北疆军权**。
恶狗与疯犬，从开篇第 4 章的一口咬痕，走到并肩问鼎——「圣旨救不了我的兵，朝廷喂不饱我的马……**我要翻过那座山，我要为自己一战**。」翻过所有的山之后，皇帝大婚办在**离北**的草野上，在天地间拜堂——**大婚 · HE**（大婚番外）。醉倒狂歌中，将进酒，杯莫停。

## 五、主要人物表（订正）

**主角**
- **沈泽川**（字兰舟）：建兴王第八子，茶石关唯一活口。昭罪寺蛰伏 → 锦衣卫镇抚 → 与萧决裂经营中博 → 终成摄政权臣。冷静隐忍、权谋深沉，为父翻案、为己而战。
- **萧驰野**（二郎）：离北王次子。骁骑悍将、封定都侯 → 叛出阒都 → 统军定天下。不羁善饮，把那只咬他的病秧子看成唯一移不开眼的人。

**离北 / 启东**：萧既明（离北世子）、陆亦栀（世子妃）、陆广白（大将）、左千秋（老将）、朝晖/晨阳（近将）；戚时雨、**戚竹音**（启东大帅）。

**昭罪寺 / 师门 / 内廷**：**齐惠连**（齐太傅，帝王之术）、**纪纲**（师父，端州烧痕，涉沈卫旧案）、纪暮（亡兄）、纪雷（锦衣卫，下狱）、**潘如贵**（掌印权阉，秋猎后下狱）、福满（掌印之争）。

**沈泽川一系**：**姚温玉**（元琢，谋士，中毒）、**奚鸿轩**（吏部/军粮案，亦敌亦用）、**费盛 / 乔天涯**（锦衣卫近卫）、**丁桃**（近侍）、**葛青青**（百合线要角，替沈办事）、**高仲雄**（学生领袖→追随）、**海良宜**（内阁阁老，后转向）、**岑愈**（内阁）、**余小再**（监察御史/文士）、**王宪**（户部主事）、**梁漼山**（崇深大人，理财）、罗牧（茶州道人）、周桂/雷常鸣/雷惊蛰/历熊/澹台虎（中博诸将）。

**皇室 / 世家 / 花党**：**咸德帝** → **李建恒（天琛帝）** → **李剑霆（女帝储君·终薨）**、**淳圣帝＝沈泽川**（登基称帝·改国号大靖）、**风泉**（李剑霆影子）、太后、**花香漪**（后与戚时雨大婚）、**花思谦**（花党）、**薛修卓**（延清，反转迎储君）、薛修易、**韩丞**（被摘牌）、孔氏、潘祥杰/潘蔺/潘远、费适、魏怀古（军粮案黑手）、杨诚（厥西布政司）。

**边沙 / 格达勒**：**白茶**（沈卫旧案，边沙女子）、**哈森**（边沙首领）、胡和鲁（长鹫部）、查干（有熊部）、边沙十二部。

## 六、与游戏版的关系

游戏《将进酒·暗阁》为**便于互动体验**，将上述脉络提炼为**十卷可玩案卷**（茶石关疑案→昭罪寺→离北来使→宫宴风云→雪夜密会→都察院/朝堂→中博疑云→端州疫病/治世→九年旧案→醉倒狂歌）。为游戏节奏做了合并与简化，**与原著严格顺序略有出入**（例如把「锦衣卫查案」概括为「都察院/查院」线，把纷繁的阒都诸案凝练为若干关键案卷）。若需要，我可以按本订正版**逐卷把游戏情节改写得更贴合原著**。
